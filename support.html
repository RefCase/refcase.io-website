Getting Started

What is RefCase?

RefCase is a reference management platform built for SaaS sales reps. It lets you organize, search, and surface customer case studies and reference accounts so you can move faster in deals — attaching the right proof point to the right prospect at the right time.

How to import your reference data

RefCase supports uploading a CSV file containing your customer reference cases. Your CSV should include columns such as:

  • Company name — the customer account
  • Industry / Segment — for filtering by vertical
  • Use case — what problem was solved
  • Outcome / Result — quantified results or outcomes
  • Reference type — case study, logo, live reference call, etc.

Once uploaded, RefCase will index your data and make it instantly searchable with live filters, grid/list views, and pitch generation.

Using search & filters

Use the search bar to find references by keyword — company name, industry, outcome, or use case. You can combine search with the category filters to narrow results by segment, deal size, or reference type. Toggle between grid and list view depending on how you like to browse.
